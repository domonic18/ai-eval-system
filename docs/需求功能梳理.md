## 二、核心评测系统
### 2.1 评测任务管理
任务管理功能包括：
- 评测任务创建
- 评测任务列表
- 评测任务详情(实时日志查看)
- 评测任务删除
- 评测任务终止
- 评测任务结果查看

#### 2.1.1 评测任务创建
```mermaid
graph TB
    A[开始] --> B[选择评测模型]
    B --> C{选择接入方式}
    C -->|API接入| D1[填写API地址、API Key、Model Name]
    C -->|Dify| D2[填写Dify服务地址、API Key、Model信息]
    
    D1 --> E[选择数据集]
    D2 --> E
    E --> F[配置执行参数]
    F --> G[确认任务配置]
    G --> H[任务创建完成]

```
